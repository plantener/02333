Copies an ELF image to memory and prepares a process.

prepare_process does some checks to avoid that corrupt images gets copied to memory. However, the checks are not as thorough as the check in initialize.

Returns:
A prepare_process_return_value struct holding the first address of the process image and an address to the page table for the process.
Parameters:
elf_imagePoints to the ELF image to copy.
processThe index of the process that is to be created.
memory_footprint_sizeHolds the maximum amount of memory, in bytes, the image is allowed to use.
